## 🚀 CI/CD Workflow

This portfolio is automatically deployed using GitHub Actions.

### Deployment Process

1. Code is pushed to the `main` branch.
2. GitHub Actions workflow is triggered.
3. Project dependencies are installed.
4. Production build is generated.
5. Static files are deployed to GitHub Pages.
6. Updated portfolio becomes available automatically.

This workflow eliminates manual deployment and ensures continuous delivery of updates.
